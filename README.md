# 🎓 EduAssist Neo – AI College Guide

EduAssist Neo is a smart frontend-based AI-style college guidance system that helps students compare and understand top new-gen tech colleges in India.

It provides structured insights, comparisons, and mock interview simulation in a clean interactive UI.

---

## 🚀 Live Demo
https://livedashboardsite.github.io/edu-assist/

---

## 📌 Features

### 🏫 College Intelligence System
- Detailed profiles of:
  - Scaler School of Technology
  - Newton School of Technology
  - Vedam School of Technology
  - NIAT (NxtWave Institute)

### ⚖️ Smart Comparison Engine
- Compare colleges side-by-side
- Fees, placements, scholarships, and selection process

### 💰 Fee & Scholarship Analysis
- Fee ranking system
- 100% scholarship detection

### 🎤 Mock Interview System
- HR + Technical + Aptitude questions
- Auto scoring (1–10)
- Feedback generation system

### 💡 AI-style Chat Interface
- Rule-based intelligent response engine
- Keyword detection system
- Context-aware replies

### 🎨 Advanced UI/UX
- Animated landing page
- Canvas-based visual effects
- Responsive chat interface
